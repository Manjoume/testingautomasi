Cara Deploy ke Vercel (Langkah Lengkap)
API key Anthropic tidak perlu diisi di sini — disimpan aman di server Vercel sebagai environment variable, tidak terekspos ke browser.

A. Siapkan File

  1. Download semua file yang sudah diberikan. Pastikan struktur foldernya benar:
    inspeksi-slf/
      ├── index.html
      ├── vercel.json
      ├── google-apps-script.js
      └── api/
          └── analyze.js

B. Buat Akun & Deploy ke Vercel

  1. Buka vercel.com → klik Sign Up → pilih Continue with GitHub (buat akun GitHub dulu jika belum ada, gratis)
  2. Setelah login Vercel, klik Add New… → Project
  3. Di halaman Import, cari tombol "upload" atau "Browse" di bagian bawah halaman → upload seluruh folder inspeksi-slf
  4. Vercel akan mendeteksi project. Sebelum klik Deploy, buka bagian Environment Variables (bisa scroll ke bawah di halaman konfigurasi)
  5. Tambah environment variable:
    Name: ANTHROPIC_API_KEY
    Value: sk-ant-api03-xxxxx (dari console.anthropic.com)
    Klik Add
  6. Klik Deploy — tunggu 30–60 detik
  7. Vercel memberikan URL seperti inspeksi-slf.vercel.app — ini URL aplikasi Anda, bagikan ke seluruh tim

C. Dapatkan API Key Anthropic

  1. Buka console.anthropic.com → daftar / login
  2. Klik API Keys di menu kiri → Create Key
  3. Beri nama key (contoh: inspeksi-slf) → klik Create Key
  4. Copy key yang muncul (dimulai dengan sk-ant-api03-...) — hanya tampil sekali
  5. Paste ke Vercel Environment Variables seperti langkah B.5 di atas

D. Setup Google Sheets (Opsional)

  1. Buka Google Sheets baru di sheets.new
  2. Klik menu Extensions → Apps Script
  3. Hapus semua kode yang ada → paste seluruh isi file google-apps-script.js
  4. Klik Deploy → New Deployment
  5. Type: Web app | Execute as: Me | Access: Anyone → klik Deploy
  6. Klik Authorize access → pilih akun Google → Allow
  7. Copy URL yang muncul → paste ke tab Pengaturan aplikasi ini → Simpan
Jika ingin mengganti API key di kemudian hari: buka Vercel Dashboard → Project → Settings → Environment Variables → edit nilai ANTHROPIC_API_KEY → Redeploy.
